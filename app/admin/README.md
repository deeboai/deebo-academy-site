# Admin Routes

## Purpose

This folder contains the Academy administrator workspace.

## What Lives Here

- dashboard entry page
- workflow queue
- route layout and shared shell
- intake review
- contacts and students
- tutors
- sessions
- payments
- session notes
- portal access management
- email operations
- system health

## Notes

All pages in this folder require authenticated Academy admin access.
`/admin/workflow` is the main operational queue.
`/admin/access`, `/admin/emails`, and `/admin/system` are the main control surfaces for account lifecycle, email
operations, and deployment health.
