# Workflow Route

## Purpose

This folder contains the admin workflow queue for the Academy critical path.

## Responsibilities

- show the next blocked operational steps in order
- keep intake, conversion, assignment, scheduling, note review, and recap follow-through in one queue
- expose sessions waiting on tutor notes and validated notes waiting on recap send
- reduce the need to navigate across multiple admin sections just to find the next action

## Notes

This route is intentionally queue-driven rather than entity-driven.
