# Library Helpers

## Purpose

This folder contains non-visual application logic.

## Major Areas

- auth and access control
- Academy data helpers
- Academy workflow, system, audit, and portal-scoped helpers
- Academy status definitions
- input validation
- email delivery
- Stripe and Google Calendar integrations
- Supabase clients

## Notes

If a concern is shared across routes and is not itself a component, it usually belongs here.
