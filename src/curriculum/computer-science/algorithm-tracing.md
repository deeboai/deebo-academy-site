## Tutor Notes
Students move too quickly through conditionals and loops. Slow them down enough to record state changes line by line.

## In-Session Examples
Trace an if-else block, a while loop, and a for loop with counters and accumulation.

## Common Mistakes
Students skip iterations, misread comparison operators, and lose track of variable updates inside the loop body.

## Practice Problems
Trace three short programs and explain the final value of the main variable in each one.

## Homework
Annotate loop execution tables for two additional tracing examples and write one paragraph explaining the control flow.

## Answer Key
Correct answers list each variable update in order and explain why the loop stops.
