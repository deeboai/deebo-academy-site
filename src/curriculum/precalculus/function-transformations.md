## Tutor Notes
Anchor the lesson around parent functions first. Only introduce multiple transformations together after the student can interpret single shifts and stretches accurately.

## In-Session Examples
Graph y = x^2, y = (x - 3)^2 + 2, y = -2|x + 1|, and y = 0.5(x + 4)^3 - 1.

## Common Mistakes
Students frequently reverse horizontal shifts and forget that outside coefficients affect vertical behavior.

## Practice Problems
Describe the transformation from each parent function and sketch the resulting graph.

## Homework
Translate five equations into graph descriptions and five graph descriptions into equations.

## Answer Key
The key should show direction, amount, and stretch or reflection for each transformation.
